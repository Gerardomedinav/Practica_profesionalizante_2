DIAGRAMA DE CONTEXTO NIVEL 0

El sistema de Estructura se representa mediante un Diagrama de Contexto (Nivel 0) que muestra
las interacciones del sistema con su entorno. Este sistema consta de un único proceso y está
relacionado con agentes externos.

El sistema tiene la capacidad de realizar diversas funciones, como el registro de Usuarios,
Categorías de Productos, Clientes y devoluciones. Además, puede realizar la Facturación de las
ventas y generar reportes en formato PDF sobre productos, clientes y ventas. Estos reportes
permiten obtener estadísticas de las ventas realizadas en un determinado período de tiempo.
El objetivo principal de este sistema es optimizar, gestionar y controlar los procesos
administrativos y la información financiera de manera eficiente. Esto facilita la toma de
decisiones precisas y oportunas. Además, el sistema permite generar balances de pérdidas y
ganancias en tiempo real, así como obtener información sobre ventas diarias, ventas por
productos, ventas de servicios, compras, devoluciones y proveedores.

El Diagrama de Nivel Superior (Nivel 1)

El Diagrama de Nivel Superior (Nivel 1) del sistema muestra los procesos y su relación con el almacenamiento y las entidades externas. En este nivel, los procesos no se relacionan entre sí. El sistema cuenta con diferentes procesos que permiten realizar diversas funciones. Estos procesos incluyen el registro de Usuarios, Categorías de Productos, Clientes y devoluciones. Además, el sistema es capaz de realizar la Facturación de las ventas y generar reportes en formato PDF sobre productos, clientes y ventas. Estos reportes son de gran utilidad, ya que proporcionan estadísticas detalladas sobre las ventas realizadas en un período de tiempo específico. De esta manera, se obtiene información relevante para la toma de decisiones precisas y oportunas. El objetivo principal del sistema es optimizar, gestionar y controlar los procesos administrativos y la información financiera de manera eficiente. Esto permite agilizar las operaciones y facilitar la toma de decisiones estratégicas en el negocio. Además, el sistema proporciona la capacidad de generar balances de pérdidas y ganancias en tiempo real, lo que brinda una visión clara de la situación financiera del negocio. También permite obtener información detallada sobre ventas diarias, ventas por productos, ventas de servicios, compras, devoluciones y proveedores.
