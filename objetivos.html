OBJETIVOS

OBJETIVO GENERAL

En primer lugar implementar un sistema automatizado de gestión de ventas, inventario y
proveedores en la tienda Código Urbano para mejorar la eficiencia y precisión en el seguimiento
de las existencias, reducir los errores humanos y mejorar la atención a la demanda de productos
específicos. Además ofrecer una página web publicitaria para promocionar la tienda vía online,
para posteriormente poder realizar la venta en línea en dicha página.

Menú principal para acceder a las funcionalidades:

OBJETIVOS ESPECIFICOS

• Implementar un sistema automatizado de inventario que permita llevar un registro preciso y
actualizado de las existencias de productos en la tienda.

• Desarrollar un sistema automatizado de gestión de proveedores que permita realizar pedidos de
forma más eficiente y precisa, reduciendo el tiempo dedicado a la revisión manual de las
existencias.

• Implementar un sistema automatizado de gestión de ventas que permita registrar las ventas
diarias de forma precisa y segura, evitando errores humanos y pérdida de información.

• Capacitar al personal de la tienda en el uso del nuevo sistema automatizado para asegurar una
transición sin problemas y una adopción efectiva del mismo.

• Desarrollar una página web publicitaria para dar a conocer los productos y promociones
disponibles en la tienda, y finalmente implementar una tienda en línea.
