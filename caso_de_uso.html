“CODIGO URBANO – ROPA DEPORTIVA”

Imagenes de Diagrama de Casos de Uso




El diagrama de caso de uso en el contexto del sistema
de ventas es un diagrama de flujo que muestra los pasos y las interacciones entre los diferentes
componentes del sistema. En este diagrama, se puede ver cómo los clientes interactúan con el
sistema de ventas, desde la creación de una cuenta hasta la realización de una compra.

El diagrama comienza con el cliente, que puede acceder al sistema de ventas a través de una
página web o una aplicación móvil. Una vez dentro del sistema, el cliente puede buscar
productos, verificar el estado de su cuenta, y realizar compras. El sistema de ventas también
permite al cliente verificar el historial de sus compras y recibir información sobre sus
pedidos.
